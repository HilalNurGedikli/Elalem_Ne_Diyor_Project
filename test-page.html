Elalem Chrome Extension Test Sayfası

Bu Sayfa Hakkında

Bu sayfa, Elalem Chrome Extension'ının test edilmesi için oluşturulmuştur.

Test İçin:
1. Chrome Extension'ı yükleyin
2. Bu sayfayı açın
3. Sağ üstte çıkan "🔍 Elalem Analizi" butonuna tıklayın
4. Extension'ın API ile iletişim kurmasını bekleyin

Site Özellikleri

  • Site Adı: Test Sitesi
  • Kategori: Generic
  • Test URL'i: file:// protokolü
  • Analiz Tipi: Universal site support

Beklenen Sonuç

Extension bu sayfada şunları yapmalı:

  • Sayfa bilgilerini çıkarmalı (başlık, URL, vb.)
  • Python backend'e POST isteği göndermelidir
  • Backend'den analiz sonucu almalıdır
  • Sonucu sayfada göstermelidir
